Dynamic Image

To adapt images to the screen size, use <picture>.

  • After using the tag mentioned above, put an image normally, using <img>;
  • For creating an adapted image use <source> above the image placed;
  • media="" paramether defines the screen size to adapt the image;
  • srcset="" paramether is used to write the picture source;
  • type="" paramether defines the media type;

Audio

Locally Hosted

To put audios on your site use <audio> tag and the next paramethers.

Uncompatible Way

  • src="" paramether is used to write the audio source;
  • controls paramether gives the audio media control;
  • autoplay paramether, as itself says, automatically plays the audio;
  • loop paramether autoreplay the audio;
  • preloads="" is basically used to defines when media is loaded:
    • "auto" load audio at same time the page is loaded, slowing down the page load;
    • "metadata" load some audio infos at same time the page is loaded;
    • "none" doesn't load any information while the page is loaded.

Compatible Way

The compatible way to put an audio in your site is used to turn avaible different files formats (MP3 - MPEG -, WAV, OGG). To use that way make the same in above way, however, does not write a single source. Use <source> into <audio> tag as done in dynamic pictures, write the different sources and put a link in case the files formats aren't working, wich will redirect to the audio download.

Videos

  • To search and download videos, use Pexels;
  • To convert videos for free use HandBrake, the same way Gimp is used for photos;
  • Pay attention in video size and compatibility;
  • Video formats may be:
    • MP4 (.mp4 and .m4v);
    • WebM (.webm);
    • OGG (.ogv).

Locally Hosted Videos

  • Do the same as when adding audios but, using video tag;
  • The width="" paramether allows you to define the video width on screen;
  • poster="" is an paramether used to write the thumbnail source.

Externally Hosted Videos

We can integrate videos from YouTube or other platfoms like Vimeo. Just choose the video, go to "Share" and use the "embed" option.
