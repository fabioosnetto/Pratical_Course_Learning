CSS Inline

The least used way of writing styles using CSS3 is writing in each line, adding the styles as a paramether of the current tag.

  • Choose the tag to customize;
  • Add the style paramether;
  • Write the customizations.

Code used to write the above part (CSS inline example):

      
   <body style="background-color: black; color: white; font-family: 'Times New Roman', Times, serif;">
      <h1 style="background-color: brown;">CSS Inline</h1>
      <p>The least used way of writing styles using CSS3 is writing in each line, adding the styles as a paramether of the current tag.</ p>
      <ul style="background-color: darkslategrey;">
         <li>Choose the tag to customize;
         <li>Add the <code>style</code> paramether;
         <li>Write the customizations.
      </ul>
   </body>

Local/Internal CSS

Knowing that inline CSS isn't the best option to write the styles of your website, we can a better way: Internal CSS. With that way to writing styles, the customizations aren't written in each line, but rather, written on head scope.

  1. Open and close the <style> tag;
  2. Inside the "style" tag, write the tags to be customized;
    • Don't use "<" and ">";
    • Open and close braces ("{}");
  3. Inside the braces of the tag, write the customizations, as done in CSS inline.

Using that way, the code is more organized and we don't need to write the same thing for multiple tags. However, for especific customizations, we still need to write the inline way, until we learn a way to better specify the tags.

External CSS

The better way to write the CSS styles is surely using the external way. That will allow us to write a single code that can be used for multiple codes.

  1. Create a ".css" file;
  2. Write @charset "UTF-8";
  3. Write the same way it was written in local CSS;
  4. Use "link:css" to include the styles of the ".css" file.
