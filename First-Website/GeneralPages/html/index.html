Main Formats

Bold/Highlight

For a semantic language, such as HTML, we talk about Highlighted Words and not words in bold (shape language).

Bold: <b>;
Highlight: <strong>.

Italic/Emphasis

We use Italic for a shape language, such as CSS, but for a semantic language we use Emphasis.

Italic: <i>;
Emphasis: <em>.

Mark

We also can mark a text semantically using <mark>.

Big and Small Text

Using <big> we can write a big word for a shape function and using <small> we can write a small word semantically.

Deleted and Inserted Text

For writing semantically we can mark a deleted text using <del> and mark a inserted text using <ins>.

Superscript and Subscript

Writing a mathematical exponent we use <sup>, as in 3366 + 62.8 and writing a molecular formula we use <sub>, as in KNO3.

Codebase - Preformat

  1. To create an internal link you need to do almost the same when you're creating an external link, using the <a> tag;
  2. Unlike external links, you don't need to write full link and open it in othe tab;
  3. Use rel="next" paramether to semantically says that the link is the next page of the current tab;
  4. Use rel="prev" paramether to semantically says that the link is the previous page of the current tab;
